# BakeCake — Telegram Bot для заказа тортов

Telegram-бот для кондитерской «BakeCake», позволяющий пользователям заказывать готовые торты или создавать кастомные торты с различными опциями.

## Функциональность

**Для пользователей:**

- Регистрация с согласием на обработку персональных данных
- Просмотр каталога готовых тортов с изображениями
- Конструктор тортов с выбором:
  - Количество уровней (1-3)
  - Форма (круг, квадрат, прямоугольник)
  - Топпинг (7 вариантов)
  - Ягоды (4 варианта)
  - Декор (6 вариантов)
  - Надпись на торте
- Корзина (добавление/удаление товаров)
- История заказов
- Промокоды для получения скидок
- Выбор даты и времени доставки (с учётом рабочего времени пекарни)
- Комментарии к заказу

**Для администраторов:**

- Просмотр всех пользователей
- Просмотр всех заказов
- Смена роли (для тестирования)

## Технологии

- **Python 3.10+**
- **python-telegram-bot** 22.3.\* — взаимодействие с Telegram API
- **JSON** — хранение данных в демо-режиме
- **python-dotenv / environs** — работа с переменными окружения
- **Django** (опционально, для будущей интеграции с реальной БД)

## Установка и запуск

1. Клонируйте репозиторий

```bash
git clone <repository-url>
cd cake-tg-bot
```

2. Установите зависимости

```bash
pip install -r requirements.txt
```

3. Создайте файл `.env` в корне проекта

```env
TG_BOT_TOKEN=ваш_telegram_bot_token
LEVEL_1_PRICE=400
LEVEL_2_PRICE=750
LEVEL_3_PRICE=1100
EVENING_HOUR=18
WORK_HOURS_START=8
WORK_HOURS_END=21
MIN_PREPARATION_TIME_HOURS=2
MIN_DELIVERY_TIME_HOURS=1
```

4. Подготовьте статические файлы

- Создайте папку `static/images/`
- Поместите в неё изображения тортов, пути к которым указаны в cakes.json (например, `static/images/berry_cake.jpg` и т.д.)
- Положите файл `personal_data_consent.pdf` в папку `static/` (можно создать пустой PDF-файл)

5. Сгенерируйте тестовые JSON-данные (если папка `demo_data/json/` пуста)

```bash
python demo_data/create_test_data.py
```

6. Запустите бота

```bash
python main.py
```

## Особенности реализации

**Кастомизация тортов**

- Обязательные поля: количество уровней, форма, топпинг
- Дополнительные опции: ягоды, декор, надпись
- Стоимость рассчитывается автоматически на основе выбранных компонентов

**Система доставки**

- Учитывается рабочее время пекарни (8:00–21:00)
- Минимальное время приготовления — 2 часа
- Минимальное время доставки — 1 час
- Наценка 20% за доставку в течение 24 часов с момента заказа

**Безопасность**

- Согласие на обработку персональных данных
- Валидация вводимых данных (имя, телефон, адрес)

## Реализовано:
